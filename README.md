# deep-thoughts

This is a simple social media full stack application. Where users can create thoughts, add reaction to thoughts, and also add friends to their friend list.


## Usage

* React
* Node
* NoSQL
* GraphQl
* Express
* MongoDB
* MongoDB Atlas
